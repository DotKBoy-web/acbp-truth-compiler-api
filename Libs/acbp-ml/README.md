# ACBP ML

ACBP ML is the machine-learning adjacent layer for the Al-Anazi Categorical-Boolean Paradigm.

It provides:

- numeric feature binning
- categorical feature normalization
- ACBP symbolic classifier
- smooth-risk policy scoring
- prediction explanation
- confusion-matrix evaluation

ACBP ML is not a replacement for machine learning. It is an interpretable truth-space and policy layer that can be used beside ML models.
