# Marketplace Listing Draft

## Name

ACBP Truth Compiler API

## Subtitle

Compile categories, flags, dashboard states, and workflow logic into auditable declared-truth artifacts.

## Short Description

ACBP Truth Compiler API helps developers, analysts, and BI teams expose the hidden truth structure behind dashboards, categorical data, workflow states, and rule-based systems.

## Full Description

ACBP Truth Compiler API is a lightweight plugin API for declared-truth modeling.

It helps calculate truth-space size, declared valid states, derived invalid states, truth density, compact feature recommendations, and dashboard result equivalence.

Use it before dashboards, semantic layers, AI agents, or analytics pipelines to detect state explosion, weak feature design, invalid categorical combinations, and inconsistent dashboard outputs.

## Use Cases

- Dashboard logic validation
- BI semantic layer checks
- Categorical data quality
- ML feature compactness screening
- Workflow state validation
- Live SQL vs compiled-state comparison
- AI agent guardrails for structured truth
- Healthcare operations dashboard governance
