What the API does

Most analytics tools visualize results. ACBP Truth Compiler exposes the declared truth structure behind those results.

Compile

Truth-space artifacts

Calculate possible states, declared valid states, derived invalid states, and truth density from structured categories.

Reduce

Feature compactness

Screen categorical and binned features to avoid uncontrolled state explosion in dashboards, models, and agents.

Validate

Dashboard equivalence

Compare Live SQL outputs against compiled-state outputs to detect semantic drift, missing states, and inconsistent logic.

Core endpoints

The API is intentionally small, plugin-ready, and JSON-first for fast integration.

POST /v1/truth-space/compile

Compile declared truth specs

Returns truth-space size, declared valid count, derived invalid count, density, and deterministic brief.

POST /v1/features/compact

Recommend compact feature sets

Identifies useful categorical/binned dimensions while avoiding IDs, timestamps, and raw continuous measures.

POST /v1/dashboard/compare

Compare dashboard outputs

Checks semantic equivalence between Live SQL result sets and compiled-state result sets.

GET /v1/clinical-dashboard/spec

Clinical operations spec

Provides a structured example for healthcare operations dashboard governance and state validation.

Example request

Object and color example: Apple can be red or green, Banana can be yellow or green, Mango can be red or yellow. 

{
  "name": "object_color",
  "dimensions": [
    { "name": "TargetClass", "values": ["Apple", "Banana", "Mango"] },
    { "name": "Color", "values": ["Red", "Yellow", "Green"] }
  ],
  "declared_truths": [
    { "TargetClass": "Apple", "Color": "Red" },
    { "TargetClass": "Apple", "Color": "Green" },
    { "TargetClass": "Banana", "Color": "Yellow" },
    { "TargetClass": "Banana", "Color": "Green" },
    { "TargetClass": "Mango", "Color": "Red" },
    { "TargetClass": "Mango", "Color": "Yellow" }
  ]
}
