Core API Endpoints

POST /v1/truth-space/compile

Compile declared truth specs and return auditable truth-space metrics.

POST /v1/features/compact

Screen column metadata or records and recommend compact truth dimensions.

POST /v1/dashboard/compare

Compare Live SQL and compiled-state result sets for semantic equivalence.
